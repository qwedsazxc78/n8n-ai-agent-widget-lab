M8n 基礎功能

可視化工作流設計

拖拉節點即可打造流程，支援條件分支、迴圈與子流程。

Webhook 觸發器

輕鬆建立 HTTP/REST 端點，接收事件驅動。

排程與佇列

內建 Cron 與佇列，確保任務準時執行。

200+ 節點整合

涵蓋 Slack、Supabase、MySQL、GCP、AWS 等常用服務。

認證管理

統一管理 API 金鑰與 OAuth 資訊，安全又方便。

版本控制

支援 Git 匯出 / 套件化部署，輕鬆在多環境同步。
